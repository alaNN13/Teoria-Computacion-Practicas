# Teoria-Computacion-Practica-1
This is a desktop app, that was developed in PYTHON programming language through FLET library. The program makes fundamental operations about alphabets & text strings using a graphic intuitive interface.

##MAIN FEATURES
**Stirngs Operations**
INCLUDES CALCULATION AND SAMPLE OF:
*Prefixes of a string.
*Sufixies of a string.
*All possible susbstrings.
*If is necessary, prints empty string (**ε**).
*It allows exporting the results in a file 'resultados.txt'.

**Klenne Closure and positive
*It allows introduce a custom alphabet separated by commas (example: `a,b` or `0,1`)
*It allows define a maximum length for the combinations.
*Generates the **Kleene Closure (Σ*)**, including the empty string (ε).
*Generates the **Positive Closure (Σ+)**, excluding the empty string.

##REQUIREMENTS
* Python 3.8 or higher.
* `flet` library installed.
* (The `itertools` library is included by default in Python).

##Instalation and execution 
Follow these steps to run the program on your local machine:

1. **Open your terminal** and navigate to the project folder.
2. **Activate your virtual environment** (if you are using one):
   venv\Scripts\activate
3.**Install the dependencies** (if you haven't already):
   pip install flet
4.**Execute the program**
   python PRÁCTICA 1_OPERACIONES BÁSICAS.py
