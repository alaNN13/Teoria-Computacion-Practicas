# Teoria-Computacion-Practica-1

This is a desktop app that was developed in the PYTHON programming language through the FLET library. The program makes fundamental operations about alphabets & text strings using an intuitive graphic interface.

## MAIN FEATURES

**Strings Operations**
INCLUDES CALCULATION AND SAMPLE OF:
* Prefixes of a string.
* Suffixes of a string.
* All possible substrings.
* If it is necessary, prints empty string (**ε**).
* It allows exporting the results to a file `resultados.txt`.

**Kleene Closure and Positive Closure**
* It allows introducing a custom alphabet separated by commas (example: `a,b` or `0,1`).
* It allows defining a maximum length for the combinations.
* Generates the **Kleene Closure (Σ*)**, including the empty string (ε).
* Generates the **Positive Closure (Σ+)**, excluding the empty string.

## REQUIREMENTS

* Python 3.8 or higher.
* `flet` library installed.
* (The `itertools` library is included by default in Python).
